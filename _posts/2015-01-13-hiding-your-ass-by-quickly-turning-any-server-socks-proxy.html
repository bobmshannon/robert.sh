A friend had shown me this quick little tip the other day. It involves the use of dynamic port forwarding to turn any server into a SOCKS proxy as long as you have SSH access to said server.

First, connect to the server you'd like to hide behind, let's use the one hosting this web site for demonstration purposes.

ssh -ND {LISTEN-PORT} {USER}@{HOSTNAME}

In my case I chose port 80, and logged into my server as root.

Finally, re-configure the program that you'd like to route through the proxy. Let's add a layer of anonymity to our web browsing and update some connection settings in Firefox.

The SOCKS Hosts will be localhost or 127.0.0.1, and the port will be 80 or the value you chose for {LISTEN-PORT} in the command ran earlier.

Finally, you may want to double check that everything is working by going to a site such as IPChicken.

.

As you can see I am now browsing the internet through my server (which resolves to 54.186.93.148 at the time of this writing).
